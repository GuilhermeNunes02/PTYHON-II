# 4.2 - API (Respostas)

**1. Afirmação FALSA:**  
**c)** APIs não podem ser usadas para manipular dados em tempo real.

**2. Afirmação VERDADEIRA:**  
**b)** É obrigatório definir os parâmetros de endpoint na URL, enquanto parâmetros de query e cabeçalho podem ser opcionais.

**3. Comando correto com `requests`:**  
**a)** `requests.get("https://api.nasa.gov/planetary/apod", params={'api_key': 'DEMO_KEY'})`

**4. Paginação em APIs:**  
**b)** Você adiciona parâmetros na query para acessar as páginas seguintes.

**5. Método comum de autenticação:**  
**b)** Enviar um token de autenticação no cabeçalho da requisição.
